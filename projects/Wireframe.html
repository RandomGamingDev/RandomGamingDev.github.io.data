The Wireframe game engine

The Wireframe game engine's a powerful, yet simple no GUI, 0-cost game engine designed in C++.
The goal for the engine is to eventually be able to support more and more APIs, libraries, and more, in a simple way
Imagine an engine, controlled by the community, that gives power to the developer, is simple enough for a large part of the developer community to understand.
Plus, being able to support multiple different platforms and their many different features due to its flexibility
If this sounds like a project that you're interested in, you can contribute by posting a pull request at the repo (here's the link to Wireframe).
